# Fast E-COM

## Description
Fast E-COM is a cutting-edge e-commerce platform designed to provide a seamless shopping experience, utilizing the latest technology for a smooth and efficient transaction process.

## Features
- User-friendly interface
- Secure payment gateway
- Multi-vendor support
- Real-time inventory management
- Order tracking

## Tech Stack
- **Frontend:** React, Bootstrap
- **Backend:** Node.js, Express
- **Database:** MongoDB

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Dhanush-M-1/Fast-E-COM.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Fast-E-COM
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
- Start the application:
   ```bash
   npm start
   ```
- Access the application at `http://localhost:3000`.
